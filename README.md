# BrainMend 脑优化

颈椎康复训练与认知评估一体化平台。通过蓝牙陀螺仪捕捉头部运动，驱动康复游戏和检测模块；同时提供 12 项认知能力评估，生成脑区功能报告。

**在线地址**: [https://fxs-84.github.io/brainmend/](https://fxs-84.github.io/brainmend/)

---

## 功能概览

### 头动追踪系统 (首页 → 头动追踪)

| 检测模式 | 说明 |
|---------|------|
| ∞ 协调性 | 跟随红点做水平/垂直/8字/垂直左/垂直右/自由轨迹运动，评估颈椎协调能力 |
| □ ROM 检测 | 前屈/后伸/左旋/右旋/左屈/右屈 6 方向活动度测量 |
| ◎ 位置觉 | 闭眼→头颈回到记忆位置→误差分析，评估本体感觉 |
| ▦ 视动训练 | 全屏条纹视动刺激(8方向)，可调速度+中心锚点 |
| ▣ 大脑垂直感知运动 | 黑白棋盘闪动(水平/垂直/混合)，脑干-前庭-视觉整合训练 |
| ▦∞ 棋盘协调 | 棋盘背景下叠加红点跟踪(3轨迹×4方向×3颜色)，90秒训练+评分 |
| 🎮 康复游戏 | 6 款体感游戏(陀螺仪控制) |

**康复游戏列表**:

| 游戏 | 类型 | 说明 |
|------|------|------|
| 射击模式 | 太空射击 | 头部瞄准→射击敌机 |
| 点头模式 | 太空躲避 | 点头控制飞船上下 |
| 小行星隧道 | 3D 隧道飞行 | 头部控制穿越障碍隧道 |
| 公路赛车 | 公路驾驶 | Yaw 换道 + Pitch 控制速度，躲避车辆 |
| 3D 太空飞行 | 自由飞行 | 三轴陀螺仪全向控制 |
| 山谷飞行 | 山谷穿梭 | 点头控制高度穿越山谷 |

**QR 码分享**: 治疗师可生成带有效期的二维码，患者扫码后进入沙箱模式完成检测，结果自动同步云端。

### 认知评估系统 (首页 → 认知评估)

| 模块 | 中文名 | 测试内容 | 分类 |
|------|--------|---------|:---:|
| attention | 注意力 | 快速找不同 | A (限时) |
| shortmem | 短暂视觉记忆 | 记住画面 | B (记忆广度) |
| memory | 文字记忆 | 数字与文字记忆 | B (记忆广度) |
| flex | 变通能力 | 灵活切换思维 | C (正确率) |
| language | 语言理解 | 语义判断 | A (限时) |
| reasoning | 推理能力 | 逻辑推理 | A (限时) |
| planning | 规划能力 | 提前规划 | D (关卡) |
| scenerecall | 场景回忆 | 特殊事件回忆 | C (正确率) |
| memorg | 记忆组织 | 复杂信息处理 | C (正确率) |
| inhibition | 自制力 | 干扰抑制 | C (正确率) |
| visual | 视觉记忆 | 复杂位置记忆 | B (记忆广度) |
| observation | 观察能力 | 细节辨别 | A (限时) |

**两种测试模式**:
- **完整测试** (12 项) — 全维度评估 + 脑区报告 + 风险评估
- **快速测试** (6 项) — 推理/场景回忆/短暂记忆/注意力/文字记忆/视觉记忆, 核心指标精简版

### 脑区功能报告

12 项测试结果映射到 10 个脑区 (左右额叶/顶叶/枕叶/颞叶/小脑)，基于 fMRI meta-analysis 文献建立稀疏权重矩阵，生成脑区功能地形图。

详见 [docs/脑区权重-文献依据.md](docs/脑区权重-文献依据.md)

## 技术栈

| 层 | 技术 |
|----|------|
| 构建 | Vite (Rollup) |
| 语言 | Vanilla JavaScript (ES Module) |
| 渲染 | Canvas 2D + WebGL (Three.js 太空场景) |
| 体感输入 | Web Bluetooth API (维特智能 BWT901 陀螺仪) |
| 语音提示 | Web Speech API (TTS) |
| PDF 导出 | jsPDF + canvg |
| 云端存储 | jsonblob.com (零配置) |
| QR 码 | qrcodejs |
| 部署 | GitHub Pages |

## 本地运行

```bash
npx serve . -p 8080
```

项目为生产构建 (Vite bundle)，不需要 `npm run dev`。浏览器打开 `http://localhost:8080` 即可。
